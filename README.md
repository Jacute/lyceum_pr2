# English
## Hello people! This is very good game JotaroVsDio!
 There is two playable units Jotaro and Dio.
 
Controls for Jotaro:
 
W - jump
 
A - move left
 
S - crouch
 
D - move right
 
l - attack
  
k - block
 
Controls for Dio:
    
↑ - jump
    
← - move left
    
↓ - crouch
    
→ - move right
    
NUM_7 - attack
    
NUM_8 - block
